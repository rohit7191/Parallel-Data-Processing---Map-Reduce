1. Unzip the HW2 folder.
2. The source code is present under the folder "Source Code".
3. Inside the folder, each program is contained in it's own directory, along with it's makefile and POM.xml
4. Open the MakeFile. Change the environment variables
5. Place the input folder in all of the respective program directories.
6. To run standalone, be in the directory where the Makefile is present and run the command, "make alone"
7. To run standalone, be in the directory where the Makefile is present and run the command, "make cloud"
